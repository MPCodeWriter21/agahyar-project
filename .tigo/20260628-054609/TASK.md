# Dockerize the project

- STATUS: CLOSED
- PRIORITY: 80
- TAGS: docker, secrets, container, enhancement

Project must be dockerized and be easily configurable and runable via a simple compose file.

The secrets must be managed via a single .env file and a `.example.env` file must be commited to thr repo with all the configuration options.

`.env` file itself must not be committed to the repo.
