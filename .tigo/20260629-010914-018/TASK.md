# Progressive Web App (PWA) support

- STATUS: OPEN
- PRIORITY: 40
- TAGS: mobile, feature

Make the application installable as a PWA on mobile devices.

- Create a manifest.json with app name, icons, theme color
- Add a service worker for offline caching
- Ensure all pages are fully responsive on mobile
- Add touch-friendly UI elements (larger buttons, proper spacing)
- Test on iOS and Android devices
- Add splash screen and app icons
