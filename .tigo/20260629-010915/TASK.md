# SEO and discoverability

- STATUS: CLOSED
- PRIORITY: 40
- TAGS: seo, marketing

Improve search engine visibility and social sharing.

- Add meta description and keywords to each page
- Generate a sitemap.xml
- Add robots.txt
- Add Open Graph and Twitter Card meta tags
- Use semantic HTML (article, nav, main, aside)
- Add canonical URLs to prevent duplicate content
- Ensure all pages have unique, descriptive title tags
