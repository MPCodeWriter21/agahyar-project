# Comments and ranking overhaul

- STATUS: CLOSED
- PRIORITY: 65
- TAGS: rating, feature, comments, enhancement

Currently, each service accepts ratings with optional comments.

What we want:
+ Services should only have comments. The same user should be able to post multiple comments and be able to reply to different comments. Admins might answer users' questions by replying to thier comments.
+ Service centers should accept ratings and comments. The star ratings for different centers can tell the users how good that center is. Comment system for centers should be similar to the Services.

This might need new HTML pages.

Also, the services pages (E.g. `/services/2/`) should be publicaly available and ready for the search engines to index.
They will still need login for posting new comments or rankings and also for suggesting the closest center. By default it should show the map for Tehran with all the centers.
