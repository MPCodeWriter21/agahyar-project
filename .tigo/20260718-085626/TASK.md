# Admin panel UX improvements

- STATUS: CLOSED
- PRIORITY: 75
- TAGS: admin panel, enhancement

In the admin panel, when adding or modifying a service center, the admin should be able to search for a place by text so that they can more easily find its location on the map.
Also, currently they can only select the coordinates on the map. Sometimes it's easier to enter the latitude and longtitude as numbers and see the selection change on the map.

These two improvements are very important.

We can use Neshan public API for searching: https://platform.neshan.org/docs/api/search-category/search/
