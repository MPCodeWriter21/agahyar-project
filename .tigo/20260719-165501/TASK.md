# Add support for login via phone number

- STATUS: OPEN
- PRIORITY: 70
- TAGS: UX, enhancement

Currently, the only way to login is with username and password.
We should accept phone number in place of usernames.
Users will be able to enter their phone/username and password to login.
Usernames must never be consistent of all numeric values (They must contain at least one character).
We could also support login via e-mail in which case we will disallow `@` in usernames.
