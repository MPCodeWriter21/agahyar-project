# Each service center might provide multiple services

- STATUS: CLOSED
- PRIORITY: 70
- TAGS: schema, admin UX, services, dataabase, service centers

Currently, it is only possible to assign a single service to a service center which does not work in practice; a single service center might provide multiple different services and it's better if we don't have to create multiple separate items in the service centers table for the same sevice center providing multiple services.
