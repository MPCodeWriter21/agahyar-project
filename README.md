\# آگاه‌یار (Aghahyār)



\*\*سامانه اطلاع‌رسانی هوشمند خدمات دولتی\*\*



\## 🎯 درباره پروژه

آگاه‌یار یک سامانه وب‌محور است که به شهروندان کمک می‌کند قبل از مراجعه حضوری به ادارات، اطلاعات کامل خدمات دولتی (مدارک، مراحل، هزینه، زمان و نزدیک‌ترین مراکز) را به زبان ساده دریافت کنند.



\## 🛠️ تکنولوژی‌ها

\- Python 3.12

\- Django 6.0

\- SQLite

\- HTML / CSS / JavaScript



\## 🚀 نصب و اجرا

برای اجرای پروژه روی سیستم خودتان، مراحل زیر را دنبال کنید:



```bash

git clone https://github.com/Fatemehmohammadganji/agahyar-project.git

cd agahyar-project

python -m venv venv

venv\\Scripts\\activate

pip install -r requirements.txt

python manage.py migrate

python manage.py createsuperuser

python manage.py runserver
